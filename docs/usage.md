# Usage Guide

This repository is organized by chapter, with each chapter containing Python scripts, tutorials, exercises, and solutions.

## Navigating the Repository

- **Python Scripts**: Found in the `scripts/` folder of each chapter. These scripts demonstrate key concepts from the book.
- **Tutorials**: Located in the `tutorials/` folder of each chapter. They provide detailed instructions on how to use the scripts and understand the concepts.
- **Data**: Any datasets used for tutorials or exercises can be found in the `data/` folder of each chapter.
- **Exercises**: Problems that test your understanding of the chapter's content are in the `exercises/` folder.
- **Solutions**: Solutions to exercises are in the `solutions/` folder, but try solving them on your own first!
